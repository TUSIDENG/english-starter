## 字母和音标

### 音标
交互式 IPA 图表：像维基百科这样的网站提供了元音和辅音的交互式图表，并为每个符号提供了音频示例，下面的：

* 词典音标：[剑桥词典](https://dictionary.cambridge.org/help/phonetics.html)等词典提供单词的音标拼写，有时还包括发音的音频。

* 发音资源：美国传统词典和 [icSpeech](https://icspeech.com/phonetic-symbols.html)，[ipachart.app](https://ipachart.app/zh-CN/vowels) 等网站提供发音指南和表格，其中包含辅音和元音的符号和示例。
